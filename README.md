📝 Task Master: A Minimalist To-Do List
A sleek, functional, and persistent To-Do list application built with Vanilla JavaScript. This was my first project focusing on DOM manipulation, data persistence, and clean UI/UX.

✨ Key Features
Data Persistence: Uses localStorage to save your tasks. Even if you refresh the page or close the browser, your tasks stay right where you left them.

Smart Interactions: * Add tasks by clicking "Add" or pressing the Enter key.

Toggle task completion with a single click.

Delete individual tasks or clear the entire list at once.

Dynamic Counter: Real-time tracking of how many tasks you have left to complete.

Responsive Design: Modern, mobile-friendly interface with smooth animations.

🧠 What I Learned
During the development of this project, I mastered several core JavaScript concepts:

The DOM API: Creating, appending, and removing elements dynamically.

Event Bubbling: Using event.stopPropagation() to prevent a delete-button click from accidentally triggering the "complete" toggle on the parent list item.

JSON Handling: Using JSON.stringify() and JSON.parse() to move data between JavaScript objects and the browser's local storage.

Functional Programming: Writing reusable functions (like addTask) that handle both user input and saved data from memory.
